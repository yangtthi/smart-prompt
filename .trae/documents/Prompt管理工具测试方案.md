# Prompt 管理工具测试方案

## 概述

本文档为 Prompt 管理工具提供完整的测试方案，涵盖单元测试、集成测试、API 测试和 UI 测试。

---

## 测试策略

### 测试金字塔

```
        /\
       /  \
      / UI \        ← E2E 测试 (少量)
     /------\
    / API   \       ← 集成测试 (中量)
   /----------\
  / 单元测试   \     ← 单元测试 (大量)
 /------------\
```

### 测试技术选型

| 层级 | 工具/框架 | 说明 |
|------|----------|------|
| 单元测试 | JUnit 4 + AssertJ | SpringBoot 标准测试 |
| 集成测试 | SpringBoot Test + MockMvc | Web 层测试 |
| API 测试 | REST Assured / Postman | 手动 API 验证 |
| 前端测试 | 手动测试 + 浏览器 DevTools | 无框架，纯静态 |

### 单元测试示例

```java
@Test
public void testCreatePrompt() {
    Prompt prompt = new Prompt();
    prompt.setName("测试Prompt");
    prompt.setContent("你好 {{name}}");
    prompt.setDescription("测试描述");
    
    Prompt saved = promptRepository.save(prompt);
    
    assertThat(saved.getId()).isNotNull();
    assertThat(saved.getVariables()).contains("name");
    assertThat(saved.getVersions()).hasSize(1);
}

@Test
public void testVersionIncrement() {
    Prompt prompt = promptRepository.findById(id).orElseThrow();
    prompt.setContent("新内容");
    promptRepository.save(prompt);
    
    Prompt updated = promptRepository.findById(id).orElseThrow();
    assertThat(updated.getVersions()).hasSize(2);
}
```

```java
@Test
public void testExtractVariables() {
    String content = "你好 {{name}}，我是 {{role}}";
    List<String> vars = variableService.extractVariables(content);
    
    assertThat(vars).containsExactlyInAnyOrder("name", "role");
}

@Test
public void testReplaceVariables() {
    String content = "你好 {{name}}";
    Map<String, String> values = Map.of("name", "张三");
    
    String result = variableService.replaceVariables(content, values);
    
    assertThat(result).isEqualTo("你好 张三");
}
```
